This project focuses on analyzing Zepto product data using MySQL. The dataset includes information such as product name, category, pricing (MRP and discounted price), discount percentage, weight, and inventory details. The primary objective is to perform data cleaning, exploration, and analysis to extract meaningful business insights. Initially stored as a single table, the data was conceptually organized into entities like Category, Product, and Inventory to improve clarity and support better data management.

The project involves several key operations including removing invalid records, converting price values from paise to rupees, and checking for missing or duplicate data. Using SQL queries, different types of analysis were performed such as filtering products based on conditions (WHERE clause), calculating aggregates like average price and total stock, and using subqueries to identify products priced above average. These steps help in understanding pricing patterns, discount distribution, and stock availability across categories.

Through this analysis, valuable insights were derived, such as identifying high-value products, categories generating higher revenue, and products that are frequently out of stock. The project also highlights how bulk items often provide better value per gram and how discounts vary significantly across categories. Overall, this project demonstrates strong SQL skills, data cleaning techniques, and the ability to derive business insights from raw data, making it a solid addition to a data analytics or database portfolio.
